Clustering the mtcars dataset with k-means

This presentation was created as the final project of the Coursera's Developing Data Products course. The project is to build an application showing that data scientists can deliver a data product that is interactive so others can alter the parameters to see changes it makes on the final result.

Tools used in this project:

  • Shiny
  • R-Presentation / Slidify

The Application

This is just a simple project that uses the mtcars dataset and use it to demonstrate how to make a data product using Shiny application that takes input parameters by users and plot a clustering result applying the k-means method.
